# Proyecto Software FJ

## Descripción
Este es un sistema para gestionar clientes, servicios y reservas para la empresa Software FJ.

## Instrucciones de ejecución

1. Clona el repositorio:
    ```
    git clone https://github.com/giraldocanchila-sketch/SoftwareFJ_grupo_434.git
    ```

2. Navega al directorio del proyecto:
    ```
    cd SoftwareFJ_grupo_434
    ```

3. Asegúrate de tener Python instalado.

4. Ejecuta el programa:
    ```
    python src/main.py
    ```

## Tecnologías utilizadas:
- Python 3.x
- GitHub
